ChatGPT Memory Sources: See Exactly What Gmail Data Powers Your Personalized Answers

OpenAI is rolling out "Memory Sources" — a transparency feature that shows users exactly which context ChatGPT drew on to personalize a response. Tap the Sources icon below any reply to see which saved memories, past chats, custom instructions, or connected Gmail data influenced the answer.

The feature arrives alongside deeper memory improvements for Plus and Pro users, where ChatGPT now more aggressively pulls context from past conversations, uploaded files, and the connected Gmail app to tailor suggestions and next steps. Users retain full control: they can delete chats, disconnect apps, disable memory, or use temporary chats that leave no trace.

→ Impact: This is OpenAI solving the "AI black box" problem for email-connected assistants. For our product, this sets a transparency bar: any AI feature that reads email should be able to explain what it used and why.
