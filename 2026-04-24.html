Google Launches Workspace Intelligence + AI Inbox in Gmail

At Cloud Next '26, Google unveiled Workspace Intelligence — a Gemini 3-powered semantic layer that reads across Gmail, Docs, Sheets, Calendar, and Chat to power agentic work. The Gmail features are the headline act: AI Inbox creates a personalized briefing that highlights to-dos and filters clutter; AI Overviews lets you ask natural-language questions and get summarized answers without opening emails.

→ Spark Impact: Google is turning Gmail into an AI-first experience with proactive briefings — the exact value prop Spark offers. The MCP server also opens the door for third-party agents to deeply integrate with Gmail, potentially commoditizing email client features.

Anthropic Opens Persistent Memory for Managed Agents

Anthropic announced that persistent memory for Claude Managed Agents is now in public beta. Agents can learn from past sessions, share what they've learned with other agents, and store memories as files on a filesystem — giving developers full control over what agents retain.

The architecture mounts memory directly onto a filesystem so Claude can use its existing bash and code execution capabilities. Compaction lets agents save context summaries that persist beyond any single session.

→ Spark Impact: This makes it dramatically easier for any company — competitors or Spark itself — to build persistent AI assistants that remember user preferences, learn email patterns, and improve at calendar management over time.
